# 📅 课表转日历工具 (Schedule to Calendar)

将学校教务系统的 Excel 课表自动转换为 `.ics` 日历文件，一键导入手机/电脑日历，不再错过任何一节课！

## ✨ 功能特性

- 🔄 **自动解析**：从 Excel 课表提取课程、时间、地点、教师信息
- 📆 **智能日期**：根据学期开始日期自动计算每一周的上课时间
- 🏖️ **节假日处理**：自动识别放假、调休，跳过假期或自动补课
- 📱 **全平台支持**：生成的 `.ics` 文件支持导入 Outlook日历等日历软件
- ⚡ **双重模式**：支持 GitHub Actions 云端自动转换，也支持本地运行

## 🚀 快速开始

### 方式一：GitHub Actions 自动转换（推荐）

适合不想配置 Python 环境的同学，每次更新课表后自动获得最新的日历文件。

0. **下载课程表**：访问南昌航空大学教务系统下载课程表 `.xls` 文件到本地
1. **Fork 本仓库** 到你自己的 GitHub 账号，建议设置为私密仓库
2. **上传课表**：将你的 `学生个人课表.xls` 文件放入 `课表日程转换/` 目录
3. **修改配置**：编辑 `Configuration.json`，设置 `Path` 的值为你的课程表文件名称
4. **提交更改**：Commit 并 Push 到 `main` 分支
5. **下载结果**：在 Actions 页面查看运行结果，下载生成的 `.ics` 文件
6. **使用**：推荐在 `outlook邮箱` 内新建一个日历，将下载的文件导入到刚刚新建的日历里。各种设备自带的日历软件都支持导入 `outlook邮箱` 内的日历

### 方式二：本地运行

适合需要频繁调试或自定义功能的同学。

#### 环境要求
- Python 3.9+
- 依赖库：`xlrd==1.2.0`

#### 安装步骤

```bash
# 克隆仓库
git clone https://github.com/你的用户名/仓库名.git
cd 仓库名

# 安装依赖
pip install xlrd==1.2.0

# 放置课表文件
# 将"学生个人课表.xls"放入"课表日程转换/"目录

# 运行转换
python Schedule.py
